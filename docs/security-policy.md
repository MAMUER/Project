# Политика безопасности и обработки персональных данных

## Соответствие 152-ФЗ

Платформа соответствует требованиям Федерального закона №152-ФЗ "О персональных данных":

1. **Получение согласия** - явное согласие пользователя на обработку ПДн
2. **Обезличивание** - данные обезличиваются для аналитики
3. **Шифрование** - хранение ПДн в зашифрованном виде
4. **Разграничение доступа** - ролевая модель
5. **Логирование** - все операции с ПДн логируются

## Меры безопасности

| Уровень | Мера |
|---------|------|
| Сеть | HTTPS, TLS 1.3, VPN для внутренних сетей |
| Приложение | JWT, CORS, защита от CSRF/XSS, rate limiting |
| Данные | Шифрование в БД, ротация ключей |
| Инфраструктура | Регулярное обновление, сканирование уязвимостей |

## Обработка биометрических данных

Биометрические данные (пульс, ЭКГ, давление, SpO2, температура, сон) обрабатываются с повышенными требованиями:
- Хранение в зашифрованном виде
- Отдельное согласие на обработку
- Возможность отзыва согласия
- Автоматическое удаление по истечении срока хранения
