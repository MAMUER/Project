# Архитектура HealthFit Platform

## Обзор

HealthFit Platform — интеллектуальная веб-платформа для персонифицированного управления физической активностью и предиктивной оценки рисков здоровью на основе анализа биометрических данных.

## Архитектурные принципы

1. **Микросервисная архитектура** - каждый компонент независим
2. **Event-driven communication** - асинхронное взаимодействие через RabbitMQ
3. **Слой машинного обучения** - Python FastAPI сервисы
4. **Импортозамещение** - использование отечественного стека где возможно

## Компоненты

### Go сервисы
- **api-gateway** - единая точка входа, маршрутизация
- **auth-service** - аутентификация, JWT
- **biometric-ingest** - прием биометрических данных
- **data-processor** - обработка данных, вызов ML
- **notification-service** - уведомления

### Python ML сервисы
- **classification** - нейросетевая классификация состояния
- **generation** - GAN для генерации программ

### Хранилища
- **PostgreSQL** - основная БД
- **Redis** - кэш, очереди
- **RabbitMQ** - асинхронные сообщения

## Потоки данных

1. **Сбор данных**: устройство → biometric-ingest → PostgreSQL + RabbitMQ
2. **Анализ**: RabbitMQ → data-processor → ML classification → PostgreSQL
3. **Генерация**: API Gateway → ML generation → PostgreSQL
4. **Уведомления**: notification-service → push/email

## Безопасность
